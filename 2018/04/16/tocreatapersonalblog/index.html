发表于 2018-04-16

前言

被杭外大佬_Hisen_ 带入坑，用的是hexo和github搭建的静态博客。一路上历经各种失败，在海森的帮助下，终于成功了哈哈哈，接下来我要用我自己总结的方法教大家如何建立一个属于自己的博客。

正文

配置环境

安装node.js
[https://nodejs.org/zh-cn/] node官网链接，下载适合自己的版本，然后一路安装即可
安装git
[https://git-scm.com] git官网链接， 同样下载完一路安装即可

安装Hexo

安装完node.js和git后，下面就需要安装Hexo了
-打开终端，执行一下命令：
$ sudo npm install -g hexo
